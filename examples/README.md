# GraphQL Nexus Examples

All of these example can be run at once:

With Yarn:

```
git clone https://github.com/graphql-nexus/schema.git
cd schema
yarn
yarn examples
```

or NPM:

```
git clone https://github.com/graphql-nexus/schema.git
cd schema
npm i
npm run examples
```

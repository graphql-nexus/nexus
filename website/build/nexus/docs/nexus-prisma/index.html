Nexus Prisma

Nexus Prisma

Docs

nexus-prisma is for bridging Prisma and Nexus. It extends the Nexus DSL t with .model and .crud making it easy to project Prisma models and expose operations against them in your GraphQL API. Resolvers are dynamically created for you removing the need for traditional ORMs/query builders like TypeORM, Sequelize, or Knex. Out-of-box features include pagination, filtering, and ordering. When you do need to drop down into custom resolvers a Photon instance on ctx will be ready to serve you, the same great tool nexus-prisma itself bulids upon.

import { nexusPrismaPlugin } from "nexus-prisma";

makeSchema({
  // ...
  plugin: [nexusPrismaPlugin()],
});

If you are still using nexus-prisma@0.3 / Prisma 1 you can find the old docs here.
